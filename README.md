## 🎯 Visão Estratégica

Landing Page desenvolvida exclusivamente para o lançamento e venda do e-book **"Do Sofá à Medalha"**, de William Lopes. O projeto substitui templates genéricos por uma **Single-Page Application (SPA)** ultrarrápida. 

O objetivo central é capturar a atenção do usuário, apresentar a autoridade do autor, quebrar objeções e guiar o tráfego de forma implacável para o checkout da Hotmart, com rastreamento completo de eventos para tráfego pago.

## ⚡ Tecnologias Forjadas

A fiação do projeto foi construída utilizando o que há de mais moderno e rápido no ecossistema front-end:

* **React (via Vite):** Motor de renderização instantânea, garantindo tempo de carregamento próximo a zero.
* **Tailwind CSS (v4):** Blindagem visual e estilização em linha. Layout limpo, responsivo e com hierarquia de cores focada em contraste (Verde-Azulado / Teal e Amarelo Conversão).
* **Vercel:** Infraestrutura de nuvem global para hospedagem com Edge Network e integração contínua (CI/CD).

## 🛡️ Funcionalidades Táticas

* **Meta Pixel Integrado:** Script de rastreamento do Facebook nativo no `<head>`, com disparo de evento `InitiateCheckout` configurado diretamente nos botões de compra.
* **Conversão Mobile-First:** Botão flutuante do WhatsApp otimizado para perseguição de rolagem em smartphones, garantindo suporte a 1-clique.
* **Integração Hotmart:** Links de pagamento diretos e validados para o checkout do produto.
* **Design Brutalista/Esportivo:** Uso de sombras densas, fontes de alto impacto e CTAs (Call to Actions) agressivos.
